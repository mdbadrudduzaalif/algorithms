## 🛠️ How to Compile and Run

To compile and run any of the files, navigate to the specific directory and use a C++ compiler that supports C++17 (e.g., `g++` or `clang++`):

```bash
# Navigate to the folder
cd searching

# Compile
g++ -std=c++17 -Wall -Wextra -O2 linear_search.cpp -o linear_search

# Run
./linear_search
```
